# 究極幻想アルテマタヌポン
『究極幻想アルテマタヌポン』は、オープンソースの将棋ソフト『やねうら王』をベースとした、コンピュータ将棋思考エンジンです。

# 技術
『究極幻想アルテマタヌポン』の技術情報については、第30回世界コンピュータ将棋選手権のアピール文書をお読みください。 http://www2.computer-shogi.org/wcsc30/
