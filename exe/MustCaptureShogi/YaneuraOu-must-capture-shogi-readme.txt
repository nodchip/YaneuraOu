
・やねうら王 取る一手将棋エディションとは？

やねうら王 取る一手将棋エディションとは、やねうら王2016 Lateからの改造で、
(合法手で)駒が取れるときは必ずとらないといけないという変則ルールの将棋です。

・取る一手エディションの遊び方

    将棋所かShogiGUIから思考エンジンとして登録して使ってください。
    実行ファイルはWindows 64bit版、CPUはAVX2用にコンパイルされています。

	GUI側では王手がかかると詰みというのは認識できませんので、王手が
　　かかった局面で投了してください。(^^;

    以下のようにファイルを配置します。

    YaneuraOu-must-capture-shogi.exe      : やねうら王 取る一手将棋エディション本体
    YaneuraOu-must-capture-shogi_ja.txt   : これを用いると思考エンジンの設定項目の表示が日本語化される。

    book/must-capture-shogi.db   : 取る一手将棋用の基本定跡(制作中)

    eval/KK_synthesized.bin        : 3駒関係の評価関数で用いるファイル(KK)
    eval/KKP_synthesized.bin       : 3駒関係の評価関数で用いるファイル(KKP)
    eval/KPP_synthesized.bin       : 3駒関係の評価関数で用いるファイル(KPP)

        evalフォルダに入れる評価関数バイナリ(上記の3ファイル)は、以下のところからダウンロード出来ます。
　　　　真やねうら王のものをとりあえずお使いください。

		近々、専用の評価関数を用意します。

			https://github.com/yaneurao/YaneuraOu
			の「やねうら王評価関数バイナリ」のところ。

    ・入玉宣言勝ちに対応しています。
    ・Ponder(相手番で思考する)に対応しています。
    ・秒読み、フィッシャールールに対応しています。
    ・最小思考時間設定に対応しています。
    ・スレッド数は思考エンジン設定で選べます。
    ・定跡の指し手がランダムに選択されます。
    ・置換表サイズは、思考エンジン設定のところで設定した値に従います。

	その他は、docs/
			https://github.com/yaneurao/YaneuraOu/tree/master/docs
	配下にある、
		解説.txt
		USI拡張コマンド.txt
	なども併せてご覧ください。
